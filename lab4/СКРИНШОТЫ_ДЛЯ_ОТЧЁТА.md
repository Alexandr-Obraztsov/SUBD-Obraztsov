# Скриншоты для отчёта ЛР4

Подставьте свои скриншоты вместо заглушек. Файлы положите в каталог **word/.doc/images/** (относительно репозитория БГУИР, т.е. `БГУИР/word/.doc/images/`) с именами ниже. После этого пересоберите отчёт (MCP `compile_report` или скрипт bsuir_report).

| Имя файла | Что снять на скриншот | Подпись в отчёте |
|-----------|------------------------|------------------|
| **fig_01_backup_chain.png** | Схема или список цепочки бэкапов: Full, Log, Diff1, Diff2, Diff3 (например, список файлов в `/var/opt/mssql/backup` после выполнения скриптов, или схема на листе). Можно скрин вывода `BACKUP DATABASE` / `BACKUP LOG` с именами файлов и параметром COMPRESSION. | Схема цепочки резервных копий: Full, Log, Diff1, Diff2, Diff3 (параметры: COMPRESSION, media set) |
| **fig_02_select_users.png** | Вывод `SELECT * FROM dbo.Users` **до** и **после** добавления User_Lab4_1 (два результата в одном скрине или два скрина склеить в один). | Вывод SELECT до и после изменений в таблице Users (добавление User_Lab4_1) |
| **fig_03_select_chats.png** | Вывод `SELECT * FROM dbo.Chats` **до** и **после** добавления Chat_Lab4_2. | Вывод SELECT до и после изменений в таблице Chats (добавление Chat_Lab4_2) |
| **fig_04_select_tasks.png** | Вывод `SELECT * FROM dbo.Tasks` **до** и **после** добавления Task_Lab4_3. | Вывод SELECT до и после изменений в таблице Tasks (добавление Task_Lab4_3) |
| **fig_05_restore_sql2.png** | Протокол восстановления на sql2: вывод команд RESTORE DATABASE (full + diff1 + diff2 + diff3) и проверочных SELECT по Users, Chats, Tasks после восстановления (актуальное состояние N). | Протокол восстановления на втором экземпляре (sql2): цепочка RESTORE и проверочные SELECT |
| **fig_06_master_backup.png** | Выполнение `BACKUP DATABASE master` (скрин вывода успешного BACKUP) и/или краткая последовательность команд для восстановления master в single-user режиме. | Резервное копирование базы master (BACKUP DATABASE master) и порядок восстановления |

После того как файлы `fig_01_backup_chain.png` … `fig_06_master_backup.png` будут лежать в **word/.doc/images/**, пересоберите отчёт — картинки подставятся в документ автоматически.
